# Cliff.toml Language Support

This extension provides syntax highlighting, snippets, completions, hover, and signature help for `cliff.toml` configuration files used by [git-cliff](https://github.com/orhun/git-cliff). The language grammar is what powers the language syntax highlighting for the monaco editor in the [Cliff Notes](https://cliff-notes.dev) and I thought why not also make it available as a standalone extension for Visual Studio Code.

![Cliff.toml Syntax Highlighting](https://raw.githubusercontent.com/camalot/cliff-notes/refs/heads/main/assets/images/extension/syntax-attributes.png)
![Cliff.toml Syntax Highlighting](https://raw.githubusercontent.com/camalot/cliff-notes/refs/heads/main/assets/images/extension/syntax-function.png)

## Features
- Syntax highlighting for `cliff.toml` files.
- Snippets for common `cliff.toml` configurations.
- Completions for `cliff.toml` keys and values.
- Hover information for `cliff.toml` keys.
- Signature help for `cliff.toml` functions and macros.
